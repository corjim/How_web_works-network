
			How Web Works Exercise


What is HTTP?
HTTP is a set of rules that govern how information is exchanged btween a web server on the internet and browser.

What is a URL? 
Defined as Uniform Resource Locator is the address of a unique resource on the internet. It is one of the key mechanisms used by browsers to retrieve published resources, such as HTML pages, CSS documents, images etc.

What is DNS?
The Domain Name System (DNS) is the phonebook of the Internet. DNS is responsible for finding the correct IP address for sites.

What is a query string?
    A query string commonly includes fields added to a base URL by a Web browser or other client application, its the part of a uniform Locator that assigns values to specified parameters.

What are two HTTP verbs and how are they different?
    Two most common HTTP verbs are;
    - POST:   request with side effects(ie, change data on server)
    -GET:   get some data from the server without side effects.

What is an HTTP request?
    It is an information request message from a client to a server over the HTTP. A client is this place is a web browser.

What is an HTTP response?
    When a client sent a request, an HTTP response is the end result of a request (eg sending back HTML/CSS/JS/etc).

What is an HTTP header? Give a couple examples of request and response headers you have seen.
    An HTTP header is a field in an HTTP request or response that contains metadata and context about the request or response.
    Example of response header are content-type, Last- modified, Set-cookie, Cache-Control.
    Example of request header are Host, user-agent, Accept, Cache_control and cookie.
